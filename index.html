Quick Reference
Register a patient
Patients → Register Patient. System prevents duplicate records (same name + DOB).
Book an appointment
Appointments → Book. Double-booking on same doctor + time is blocked automatically.
Issue a prescription
Prescriptions → Issue. Requires a linked consultation. Stock is deducted on dispensing.
Undo last action
Every action shows an Undo button in the notification toast for 8 seconds.
Print invoice
Billing → click Print on any invoice to open a print-ready view.
