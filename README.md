# 🏥 Monalisa Health System

Monalisa Health System is a lightweight, modern healthcare management interface built using HTML, CSS, and Vanilla JavaScript. It is designed to demonstrate how hospitals, clinics, and medical centers can manage daily operations through a single, clean, and easy to use platform.

The project focuses on simplicity, speed, and usability, without relying on external frameworks or dependencies.

---

## ✨ Key Features

- Secure multi role login interface for different users
- Clean and interactive dashboard with key statistics
- Patient registration and medical record management
- Appointment scheduling, tracking, and updates
- Consultation workflow and clinical notes handling
- Laboratory requests and result tracking
- Pharmacy inventory management and prescription handling
- Medicine dispensing and stock monitoring
- Billing, invoices, and payment tracking
- Staff and branch management module
- Reports and operational summaries
- Responsive design for both mobile and desktop
- Built in notifications for system updates
- Integrated chatbot support interface
- Quick WhatsApp contact access

---

## 🛠️ Technology Stack

- HTML5 for structure and accessibility
- CSS3 for layout, styling, and responsive design
- Vanilla JavaScript for application logic and interactivity

### Why this approach

This stack was chosen to keep the system lightweight, fast, and easy to maintain. It runs directly in the browser without requiring installation or build tools, making it ideal for demos, prototypes, and client presentations.
